# AI（機械学習）で音を作る（Magenta.js を利用）

## 使い方
キーボードのキーを押すと、音が鳴ります（キーと音の対応関係は以下のとおり）

- きらきら星とそれを元にした音
   - a ：　きらきら星の最初の部分そのまま
   - w ：
   - e ：
   - r ：
   - t ：
- チューリップとそれを元にした音
   - a ：チューリップの最初の部分そのまま
   - s ：
- かえるのうたとそれを元にした音
   - z ：かえるのうたの最初の部分そのまま
   - x ：
